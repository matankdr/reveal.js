## Short Introduction to Java

- JVM: Java Virtual Machine <!-- .element: class="fragment" -->
- Source code is compiled to bytecode <!-- .element: class="fragment" -->
- JVM interprets bytecode and executes it <!-- .element: class="fragment" -->
- Result: interoperability between JVM languages <!-- .element: class="fragment" -->
- Great runtime optimizations (JIT) <!-- .element: class="fragment" -->
- Many JVM vendors (e.g., OpenJDK, HotSpot, Azul) <!-- .element: class="fragment" -->


## JVM Ecosystem

- The **JVM** is a platform, not just Java
- JVM supports multiple languages
  - E.g, Java, Kotlin, Scala, Groovy
  - Interoperability

> Learning Java well improves your understanding of *all* JVM languages
