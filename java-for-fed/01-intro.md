## Java Today

- Actively evolving (6-month release cycle)
- Backed by a massive ecosystem
- Used heavily for:
  - Backend services
  - Distributed systems
  - High-throughput applications
  - Long-running, mission-critical software


## Java today emphasizes

- Immutability
- Strong typing
- Structured concurrency
- Declarative style (Streams, pattern matching)

## (Very) Short Intro To JVM
- JVM: Java Virtual Machine <!-- .element: class="fragment" -->
- Source code is compiled to byte-code <!-- .element: class="fragment" -->
- JVM interprets compiled bytecode and executes it <!-- .element: class="fragment" -->
- Result: interoperabilty between JVM languages <!-- .element: class="fragment" -->
- Great optimizations in runtime (JIT) <!-- .element: class="fragment" -->
- Many JVM vendors (e.g., OpenJDK, HotSpot, Azul) <!-- .element: class="fragment" -->


## JVM Ecosystem

- The **JVM** is a platform, not just Java.
- JVM supports multiple languages
  - E.g, Java, Kotlin, Scala, Groovy
  - Interoperability

Key takeaway: 
> Learning Java well improves your understanding of *all* JVM languages
